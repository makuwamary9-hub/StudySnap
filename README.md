🎓 Student Productivity App (MVP)

## 📌 Overview
This is a simple student productivity app designed to help students manage their academic life more effectively.

## 🚀 Features
- Task and assignment tracking
- Study schedule planner
- Reminder system for deadlines
- Simple and clean interface

## 💡 Problem It Solves
Students often struggle to stay organized with assignments, deadlines, and study time. This app provides a simple solution to manage everything in one place.
